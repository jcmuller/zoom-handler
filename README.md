**Table of Contents**
- [Installation](#installation)
- [Usage](#usage)


## Installation
```bash
$ go get -u github.com/jcmuller/zoom-handler
```

## Usage
```bash
$ zoom-handler https://[group].zoom.us/j/[MEETING ID]
```
